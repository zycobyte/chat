Settings

Chat Settings

The chat settings may look difficult but once you get the hang of it, it's pretty easy.

Personal Chat Settings

Here you can select whether you are sent notifications from specific channels, or being mentioned.

Global Chat Settings

Global Chat Settings are the settings that will change things that will have an effect on other users in the chat.


First of all is the Name and Description. This simply changes the text displayed to users. Although having a description is optional, it is recommended having both be relevant to your chat because users may have many chats and not remember which chat is which if its just a generic name like 'chat here' with no description. It also helps users when they are searching.

Next is 'Generate Invite' and 'Manage Invites'. Generate Invite is a button that can be clicked which allows users to generate their own invites without being able to manage the chat. Manage Invites is where the managers can add or delete invites or view details on the ones that exist.

Enable Join Message is an optional message that is automatically sent whenever anyone joins the chat. You are able to select which channel it is sent to and what it says. It also has custom variables like $user which is replaced with the username and $chat which is replaced with the chat name.

Maintenance Mode is a mode you can turn on if you want to temporarily stop users from accessing your chat. It displays them with a message that you set, and will activate as soon as you save your settings. By default only the owner will be allowed to access the chat, however users can be added to the 'maintainers' list if you want them to be able to help you.

Roles is a list of all the roles on your server. You are able to create, remove or edit the roles, and change their settings. You can edit the name and colour, which change the name and colour. Changing the Icon will change what appears next to the users names, and the priority is how high the role it. Things like colour are set for a user from their highest role. Changing the permissions changes what the users are able to do on the server.

Channels is a list of all the channels and categorys in the server. This is the section that allows you to add, remove, or edit them by changing their position or their names.

User Settings allows you to search users and change their rank, or perform actions on them by opening their profile such as ban, kick, mute or simply send them a DM or friend request.

Account Settings

Account settings is separated into 2 sections, the first requires you to put in your password to edit, and allows you to change your username, email and password.

The second section allows you to change your notification settings, and message scanner option.

The message scanner is what adds a little colour next to any messages sent, and determines whether they could be someone asking for personal info. You can set this setting so as none, only some, or all sent messages are scanned.

Underline is done with two underscored __Like this__, and in the event you want to type '*hi*' without italics, you can put a '\' before each character \*Like this\* (the backslash will disappear for you and works with bold and underline too!)

Other Users

If you click on a users name or profile picture, it will open a menu screen allowing you to send them a friend request, block them, start a DM etc.

While you are waiting for a user to accept your friend request, there profile will appear in your 'pending' list. Once they accept it it will move to your 'friends' list and you can add them as a close friend (which will appear in both lists).

Users who send you a friend request will also appear under your 'pending' list, and you will have the option to either add them or not to add them.

You can also block users who are being mean to you. Simply hit block and Eien.no Chat will hide your messages from them and theirs from you! And you can always unblock them just as easily if you didn't mean to do it.

By starting a DM with a user, you can chat with them in a private place. This can also be done with the profile, but it can also be done by searching their name into the 'Start DM' menu too!
